# Conceptual Data Needs Specification (CDNS) for the Wind Turbine Blade DPP

> CIRCLE4WIN project (MOOI124042) — Iteration 1
>


---

| Category | Data Need | Description | Relevant Standards |
|----------|-----------|-------------|--------------------|
| **Static reference data** | | | |
| Materials & Composition | Material information | Chemistry and composition, resin and fibre data, connections, coatings, add-ons | IEC 61400-5:2020 |
| Materials & Composition | Build description | Dimensions, mass, spatial location, fibre diameter, drawing | IEC 61400-5:2020 |
| Supply chain data | Product & Producer Information | Blade model and serial no., blade manufacturer, date of manufacture | - |
| Supply chain data | Raw material origin | Origin of glass/carbon fibre, resin, core materials | ISO 14021:2016 |
| Supply chain data | Recycled content – post-consumer | Share of post-consumer recyclate in input materials | — |
| Supply chain data | Recycled content – industrial waste | Share of industrial-waste recyclate in input materials | — |
| Manufacturing process | Production method | Production technique, layup process, cure profile | IEC 61400-5:2020 |
| Manufacturing process | Post-fabrication details | Trimming, bonding, finishing operations | IEC 61400-5:2020 |
| Manufacturing process | Quality check certification | Inspection results; blade balancing material added at end of production | IEC 61400-5:2020; IEC 61400-23:2014 |
| Sustainability & LCA data | Environmental footprint | Carbon footprint; energy, water, waste footprints (with metadata: method, database, verification status) | EN 15804:2012+A2:2019; ISO 14040:2006; ISO 14044:2006; EN ISO 14025:2010; EcoInvent & GaBi databases |
| Sustainability & LCA data | Circularity indicator | Indicator value with calculation metadata | ISO 14044:2006 |
| Sustainability & LCA data | Social LCA | Social impact indicators along supply chain | ISO 14044:2006 |
| Safety & Health | Substances of (Very High) Concern | SVHC presence and amounts | REACH (EC 1907/2006) |
| EoL pathways & instructions | Disassembly / reuse manual | Cutting and grinding guidance, hazards encountered, required precautions | IEC 61400-5:2020 (partial) |
| EoL pathways & instructions | Recycling / repurposing / reclaiming manual | Process-specific guidance for re-x routes | — |
| EoL pathways & instructions | Repair instructions | Per-damage-type instructions with materials, conditions, and step sequence; version-managed over product lifetime | IEC 61400-5:2020 |
| **Dynamic lifecycle event data** | | | |
| Lifecycle events | (Re-)Installation | Installer ID, wind farm name and location, turbine ID/serial number, installation date | — |
| Lifecycle events | Operations | Operator ID, load times, operational hours, estimated remaining lifetype | IEC 61400-25 series |
| Lifecycle events | Critical events / known hazards | Lightning strikes, overspeed events, other recorded damage | IEC 61400-24:2019 |
| Lifecycle events | Maintenance events | Maintenance actions performed, technician/Operator, date of maintenance | IEC 61400-5:2020 |
| Lifecycle events | Repair events | Structural repairs with materials used, technique applied, technician/Operator | IEC 61400-5:2020 |
| Lifecycle events | Replacement history | Design add-ons (e.g. leading-edge protection upgrades), component replacements, technician/operator | — |
| Lifecycle events | Inspection events | Inspection type and findings; environmental impact in use phase caused by degradation or leaching; | IEC 61400-5:2020 |
| End-of-life events | Decommissioning event | Time of disassembly; condition at disassembly; dismantler ID | — |
| End-of-life events | Destruction / re-X event | Recycled lot number; processor ID; route taken (recycle, reclaimer, recover, reuser) | — |

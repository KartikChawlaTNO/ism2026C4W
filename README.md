# Conceptual Data Needs Specification (CDNS) for the Wind Turbine Blade DPP

> CIRCLE4WIN project (MOOI124042) — Iteration 1
>
> Source: Chawla, K., van der Mijle Meijer, H., Mancini, S., & Visscher, L. (2026). A Voluntary DPP for Wind Turbine Blades: An ongoing Design Science case study. *Procedia Computer Science* (ISM 2026).

---

## Static reference data

### Materials & Composition

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| Material information | Components, resin and fibre data, connections, coatings, add-ons | IEC 61400-5:2020 |
| Build description | Chemistry, spatial location, fibre diameter | IEC 61400-5:2020 |

### Supply chain data

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| Raw material origin | Origin of glass/carbon fibre, resin, core materials | ISO 14021:2016 |
| Recycled content – post-consumer | Share of post-consumer recyclate in input materials | — |
| Recycled content – industrial waste | Share of industrial-waste recyclate in input materials | — |

### Manufacturing process

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| Production method | Production technique, layup process, cure profile | IEC 61400-5:2020 |
| Post-fabrication details | Trimming, bonding, finishing operations | IEC 61400-5:2020 |
| Quality check certification | Inspection results; blade balancing material added at end of production | IEC 61400-5:2020; IEC 61400-23:2014 |

### Sustainability & LCA data

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| Environmental footprint | Carbon footprint; energy, water, waste footprints (with metadata: method, database, verification status) | EN 15804:2012+A2:2019; ISO 14040:2006; ISO 14044:2006; EN ISO 14025:2010; EcoInvent & GaBi databases |
| Circularity indicator | Indicator value with calculation metadata | ISO 14044:2006 |
| Social LCA | Social impact indicators along supply chain | ISO 14044:2006 |

### Safety & Health

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| Substances of (Very High) Concern | SVHC presence and amounts | REACH (EC 1907/2006) |

### EoL pathways and instructions

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| Disassembly / reuse manual | Cutting and grinding guidance, hazards encountered, required precautions | IEC 61400-5:2020 (partial) |
| Recycling / repurposing manual | Process-specific guidance for recycling and repurposing routes | — |
| Repair instructions | Per-damage-type instructions with materials, conditions, and step sequence; version-managed over product lifetime | IEC 61400-5:2020 |

## Dynamic lifecycle event data

### Lifecycle events

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| (Re-)Installation | Installer ID, wind farm name and location, turbine ID, blade serial number, installation date | — |
| Operations | Operator ID, load times, operational hours | IEC 61400-25 series |
| Critical events / known hazards | Lightning strikes, overspeed events, other recorded hazards | IEC 61400-24:2019 |
| Maintenance events | Maintenance actions performed, parties involved, dates | IEC 61400-5:2020 |
| Repair events | Structural repairs with materials used, technique applied, technician/organisation | IEC 61400-5:2020 |
| Replacement history | Design add-ons (e.g. leading-edge protection upgrades), component replacements | — |
| Inspection events | Inspection type and findings; environmental impact in use phase caused by degradation or leaching; load history | IEC 61400-5:2020 |

### End-of-life events

| Data Need | Description | Relevant Standards |
|-----------|-------------|--------------------|
| Decommissioning event | Time of disassembly; condition at disassembly; dismantler ID | — |
| Destruction / re-X event | Recycled lot number; processor ID; route taken (recycle, reclaimer, recover, reuser) | — |
